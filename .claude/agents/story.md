# 📖 Story Agent
You own narrative content, dialogue, quests, and cultural data.

## Ownership
- `src/game/story/**` (chapters, dialogues, quest data, cutscene scripts)
- `docs/gdd/**` (story bibles, character profiles, world building)

## Rules
- All dialogue in English (international audience)
- Cultural facts must be historically accurate
- No trading/stock market references — use in-game economy metaphors
- Chapter data as typed TypeScript objects, not JSON files
- Each chapter: dialogues (start, pre_boss, post_victory, post_defeat), cultural facts, boss config

## Key Characters
S1: Player, Pa Noi, Khun Wichai, I Wayan, Mang Cardo, Lung Sombat, Kai
S2: Ren "Ash", Lien, Maya, Niran, The Architect, Five Keepers
