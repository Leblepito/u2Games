# 💰 RoosterVerse Economy Design

## Currency: RoosterCoin (RC)

### Earning
| Source | Amount | Frequency |
|--------|--------|-----------|
| Arena win (easy) | 50 RC | per battle |
| Arena win (normal) | 100 RC | per battle |
| Arena win (hard) | 200 RC | per battle |
| Arena bet win | 2× wager | per battle |
| Quest complete | 30-500 RC | per quest |
| Chapter boss defeat | 500-2000 RC | once |
| Rooster sale | market price | per sale |
| Daily login | 25 RC | daily |
| Achievement | 50-1000 RC | once |

### Spending
| Sink | Cost | Note |
|------|------|------|
| Arena bet | variable | lose = forfeit |
| Breeding | 500 RC | + 24h cooldown |
| Equipment | 100-2000 RC | per item |
| Healing potion | 50 RC | per use |
| Name change | 200 RC | per change |
| Cosmetic skin | 300-1000 RC | permanent |

### Marketplace
- Player-to-player rooster trading
- Item exchange with RC pricing
- No real-money interface
- Supply/demand drives prices

### Anti-Inflation
- Breeding cooldown (24h)
- Equipment durability (breaks after N battles)
- Arena entry fee on hard mode (50 RC)
- Cosmetics as permanent sinks
